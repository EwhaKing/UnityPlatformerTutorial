### 📌 스터디 진행 방식
- 매주 커리큘럼을 따라 강의를 수강한 후 코드를 깃허브에 올립니다.
- 강의를 학습하며 배운 내용을 문서로 정리합니다.(개인 블로그, 한글, 워드 등 자유롭게 선택)
- 스터디 정기 모임 때 학습 내용을 발표하고, 오류나 문제가 발생한 경우 이를 공유하여 해결합니다.
